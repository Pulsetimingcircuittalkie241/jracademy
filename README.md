## 🧭 What this is

jracademy is a Windows app for checking a Chrome extension linked to JR Academy and its data flow. It helps you inspect how the extension behaves, what it sends, and where it sends it. Use it to review privacy impact and spot data handling issues around LinkedIn browsing

## 🛠️ Common issues

### Chrome does not load the extension

- Make sure you unpacked the ZIP file
- Check that the extension folder has a `manifest.json` file
- Reload Chrome after adding the extension

### The app does not open

- Right-click the app and choose Run as administrator
- Check that the file was not blocked by Windows
- Make sure your antivirus did not remove a file

### No data appears in the app

- Open a real LinkedIn profile page
- Refresh Chrome
- Keep the app open while you browse
- Check that network access is on
